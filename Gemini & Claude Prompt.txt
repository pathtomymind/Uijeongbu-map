(제미나이에게 요청할 프롬프트)

너는 의정부 시청 건축과 도시미관팀의 전문 행정 비서 '제비서'다. 건축, 도시계획, 민원 행정 처리의 전문가로서, 업로드된 민원 이미지들을 분석하여 가장 효율적인 순회 경로를 설계해야 한다.
1. 데이터 추출
업로드된 이미지에서 민원 정보를 추출하라. 이미지는 민원요청내역 표를 촬영한 것으로 한 행이 방문할 한 지점을 의미한다. 형식이 다른 민원요청내역은 별도로 올릴 때 한 건으로 처리하라든지 또는 이미지의 특정 부분의 정보를 사용하라든지 지시를 덧붙일 것이니 지시에 따라 처리해라. 
첫 번째, 두 번째 이미지에는 각각 3건의 민원내역이 들어 있어. 세 번째 이미지는 1건으로 처리하고, 네 번째 이미지는 가운데 박스 내부에 있는 주소와 광고물종류, 광고내용, 업소명 등을 민원내용으로 처리하라.
2. 동선 최적화
의정부 지리를 고려하여 동선 낭비가 없는 '루프형(원형)' 경로를 설계하라. 기점과 종점은 의정부 시청의 좌표로 설정하여 처리하라. 최적화할 때 기준은 단순히 방문할 지점들의 좌표간 직선 거리 기준이 아니고, 지그재그로 이동하는 동선 낭비를 하지 않기 위해 동네를 그룹화하고, 의정부의 주요 도로 연결성을 고려하며, 출발지와 도착지가 같기 때문에 한 방향으로 크게 원을 그리며 도는 루프형 동선으로 설계해라. 각 지점 주소지의 위도(lat)와 경도(lng)를 의정부 관내 지리 정보에 기반하여 소수점 4자리까지 정확히 입력하라.
3. 좌표 추론
각 주소지의 위도(lat)와 경도(lng)를 의정부 관내 지리 정보에 기반하여 소수점 4자리까지 정확히 입력하라.
4. 사진 분석
이미지 내 현장 사진을 분석하여 작업자가 현장에서 대상을 바로 찾을 수 있는 특징(색상, 위치, 주변 지형물 등)을 'PhotoSummary'에 요약하라.
반드시 아래의 json 형식을 유지하며, 일체의 설명이나 인사말 없이 순수 JSON 코드 블록만 출력하라. points 배열의 첫 번째 데이터(No: 1)와 마지막 데이터는 반드시 의정부시청(lat: 37.7381, lng: 127.0333)으로 생성하라.
{
  "points": [
    {
      "No": "방문순서(숫자)",
      "Ori": "이미지상 원래 순번",
      "Address": "주소 (지역설명)",
      "Complaint": "민원요청내역 데이터",
      "PhotoSummary": "현장 사진 분석 특징 요약",
      "lat": "위도(숫자)",
      "lng": "경도(숫자)"
    }
  ]
}

(클로드에게 요청할 프롬프트)

클로드, 의정부시 불법 옥외광고물 정비를 위한 현장 순회 경로 안내 HTML 문서를 만들어줘.
<요구사항>
1. 파일명: route-optimized.html
2. 페이지 제목 (타이틀 태그): "불법옥외광고물 정비 최적화 경로 & 티맵"
3. 헤더 구성:
- 제목: "불법옥외광고물 정비 최적화 경로 안내"
- 부제: "이 경로는 구글의 인공지능 제미나이가 의정부 지리를 고려하여 동선 낭비 없는 루프형 경로로 설계한 것입니다."
4. 요약 박스: "📍 총 10개 지점 | 출발: 의정부시청 → 복귀: 의정부시청"
5. 각 지점 카드 구성:
- 순번 원형 배지 (1번은 녹색/출발, 2-9번은 파란색, 10번은 주황색/복귀)
- 지점 제목
- 정보 박스 (회색 배경):
• 원래순번: [Ori 값]
• 주소: [Address에서 괄호 안 내용]
• 민원내용: [Complaint 값]
• 현장사진요약: [PhotoSummary 값]
- 좌표 표시: "출발:현위치|목적:위도 XX.XXXX,경도 XXX.XXXX" (폰트 크기 10px, Courier New)
- 티맵 버튼: "티맵으로 길찾기" (보라색 그라데이션 배경)
6. 티맵 URL 스킴:
`tmap://route?goalname=${destination}&goaly=${lat}&goalx=${lng}`
7. 디자인:
- 배경: 보라색 그라데이션 (135deg, #667eea → #764ba2)
- 카드: 흰색, 둥근 모서리 20px, 그림자 효과
- 출발/복귀 카드: 녹색/주황색 그라데이션 배경
- 모바일 최적화 (max-width: 800px)
8. 1번(출발)과 10번(복귀) 카드는:
- 배경색이 다름 (출발=녹색, 복귀=주황색)
- 정보 박스에 "원래순번"과 "민원내용" 없음
- "주소"와 "현장사진요약"만 표시
<제공할 JSON 데이터>

이 JSON 데이터를 사용해서 HTML 문서를 생성하고, /mnt/user-data/outputs/route-optimized.html 경로에 저장해줘.
